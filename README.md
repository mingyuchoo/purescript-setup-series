# purescript-setup-series

## Prerequisite

- install `node`
- `npm install -g purescript`
- `npm install -g spago`


## Create a project

```bash
mkdir my-project
cd my-project
spago init
```

## Build and test

```bash
spago build
spago test
```

## Install Dependencies

```bash
spago install lists foldable-trversable
```

## Run

```bash
spago run

```

## Compiliing for the browser

```bash
spago bundle-app
```
